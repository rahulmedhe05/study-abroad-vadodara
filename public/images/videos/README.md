# Video Thumbnails Guide

## Required Video Thumbnails

Please create or download video thumbnails for the video section. Recommended size: 1280x720px (16:9 ratio)

### Required Thumbnails
1. `canada-visa-thumbnail.jpg` - Canada flag + visa theme
2. `ielts-tips-thumbnail.jpg` - IELTS exam preparation theme
3. `without-ielts-thumbnail.jpg` - Study abroad without IELTS
4. `top-countries-thumbnail.jpg` - World map with flags
5. `sop-writing-thumbnail.jpg` - Writing/document theme
6. `scholarships-thumbnail.jpg` - Scholarship/award theme

### YouTube Video Embed URLs
Replace the placeholder YouTube URLs in `components/video-section.tsx` with your actual YouTube video embed URLs.

Example:
```
"https://www.youtube.com/embed/YOUR_VIDEO_ID"
```

### Creating Custom Thumbnails
Use Canva (free) to create custom thumbnails:
1. Go to canva.com
2. Select "YouTube Thumbnail" template
3. Add your branding and text
4. Export as JPG
