# Blog Images Guide

## Required Images for Blog Articles

Please download high-quality, royalty-free images from these sources and save them with the following names:

### Featured Blog Images (1200x630px recommended)
1. `study-abroad-guide-2026.jpg` - Students at university campus
2. `top-countries-2026.jpg` - World map with study destinations
3. `ielts-guide-2026.jpg` - Student studying for exam
4. `canada-visa-guide-2026.jpg` - Canadian flag with documents
5. `scholarships-2026.jpg` - Student receiving award/scholarship
6. `study-uk-2026.jpg` - UK university or Big Ben

### Free Image Sources
- **Unsplash**: https://unsplash.com (Search: "students", "university", "education", "study abroad")
- **Pexels**: https://pexels.com
- **Pixabay**: https://pixabay.com

### Image Optimization Tips
1. Use WebP or JPEG format
2. Compress images using TinyPNG or Squoosh
3. Recommended dimensions: 1200x630px for featured images
4. Keep file size under 200KB for fast loading
